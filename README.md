# AI Resume Analyzer

AI Resume Analyzer is an AI-powered career assistance application that helps users analyze resumes, calculate ATS scores, identify missing skills, generate AI-powered feedback, build professional resumes, and prepare for interviews using Google Gemini AI.

## Features

- AI-powered Resume Analysis
- ATS Score Calculation
- Resume & Job Description Matching
- Missing Skills Detection
- AI Feedback & Suggestions
- Resume Builder (PDF & Word)
- AI Chatbot Assistant
- Interview Preparation Assistant
- AI Career Roadmap Generator
- Modern Responsive UI

## Technologies Used

- React.js
- FastAPI
- Python
- Google Gemini AI

## Installation

```bash
git clone https://github.com/Jeniber-jeya-j/AI-Resume-Analyzer.git
cd AI-Resume-Analyzer

# Frontend
cd frontend
npm install

# Backend
cd ../backend
pip install -r requirements.txt
```

## Run the Project

### Backend

```bash
cd backend
uvicorn main:app --reload
```

### Frontend

```bash
cd frontend
npm start
```
